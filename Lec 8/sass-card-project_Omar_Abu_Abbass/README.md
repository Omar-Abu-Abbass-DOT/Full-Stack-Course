# Sass Card Project

## Project Overview
A modern, responsive card component built to demonstrate the power of **Sass (Syntactically Awesome Style Sheets)**. This project focuses on modular CSS architecture using variables, nesting, mixins, and partials to create scalable and maintainable styles.

## Key Features
* **Modular Architecture:** Styles are split into logical partials (`_variables`, `_mixins`, `_card`, `_base`).
* **Responsive Design:** Adapts layout for mobile devices using custom Mixins.
* **Interactive UI:** Hover effects on the card and buttons using Sass transitions.
* **Reusable Logic:** Parametric mixins for buttons and layout utilities.
* **Consistent Theming:** Global variables for colors, typography, and spacing.
