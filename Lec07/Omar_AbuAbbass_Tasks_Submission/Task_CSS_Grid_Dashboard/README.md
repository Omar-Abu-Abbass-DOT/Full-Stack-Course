## Project Overview
A responsive dashboard homepage layout designed to demonstrate mastery of **CSS Grid** as the primary layout system. The project features a structured grid container (Header, Main Content, Footer) with a responsive card system that adapts seamlessly from desktop to mobile screens using Media Queries.

## Key Features
- **CSS Grid Layout:** Full-page grid structure dividing content into Header, Main, and Footer.
- **Responsive Card System:**
    - Desktop: 4 cards per row.
    - Tablet: 2 cards per row.
    - Mobile: 1 card per row.
- **Flexbox Integration:** Used inside components (Header navigation, Card content, Footer) for precise alignment.
- **Visual Design:** Consistent color theme using CSS Variables, shadow effects, and hover animations.
- **Semantic HTML:** Utilization of `<header>`, `<main>`, `<article>`, and `<footer>` tags.

## Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (CSS Grid, Flexbox, Variables, Media Queries)
- Font Awesome (Icons)
